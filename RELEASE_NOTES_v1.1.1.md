# Habitly v1.1.1 - UI/UX Refinement & Performance Update

**Release Date:** June 5, 2026  
**Status:** Latest Stable Release

---

## 🎨 What's New

### Major Updates

- **Notion-style Design System**: Complete redesign with Notion-inspired visual language
  - New color palette with semantic color tokens
  - Refined typography using Inter and Noto Serif fonts
  - Improved spacing, borders, and shadows
  - Enhanced light and dark mode themes

- **Improved UI Components**:
  - Modernized dashboard with cover header
  - Redesigned task input with inline editing
  - Updated task filters with active indicator
  - Notion-style task rows with drag handle visualization
  - Enhanced stats dashboard with progress bars
  - New PWA update toast with better styling

- **Authentication Pages Redesign**:
  - New Login page with improved UX
  - New Register/Sign-up page
  - Better form validation and messaging

- **Task Heatmap Page Enhancements**:
  - Redesigned calendar heatmap with intensity colors
  - Improved streak display
  - Better task management UI
  - Updated editing and deletion interfaces

### Features

- **New Register Page**: Users can now create new accounts
- **Enhanced Search**: Better task filtering with improved UX
- **PWA Support**: Improved service worker update notifications
- **State Persistence**: UI state now persists across sessions

### Performance Improvements

- Font preloading for better performance
- Optimized CSS animations and transitions
- Improved scrollbar styling
- Better rendering with staggered animations
